# 특수좀비 소환 — 40,000원

`mutant_spawn`

CDDA(Cataclysm: Dark Days Ahead)에서 이식한 특수좀비 중 하나를 무작위로 소환합니다.

| 종류 | 특징 |
| --- | --- |
| 스크리머 (Screamer) | 비명으로 주변 좀비를 끌어모으는 지원형 |
| 브루트 (Brute) | 높은 체력·공격력의 근접 강화형 |
| 로치 (Roach) | 별도 AnimSet 기반 이동/공격속도 커스텀 개체 |

소환된 특수좀비는 후원자 이름이 표시되는 **네임태그**가 머리 위에 뜨고, 등장 시 랜덤 조합된 대사(예: "시발, 브루트잖아!")를 외칩니다. 네임태그 표시 여부는 서버 설정(`Donation_MutantNameTag`)으로 켜고 끌 수 있습니다.

!!! info "부활에도 유지되는 특수좀비"

    특수좀비는 [강령술](rise-up-dead-man.md)로 부활해도 종류와 능력이 그대로 유지되도록 별도의 영속 레지스트리로 관리됩니다. 서버가 재시작되어도 마찬가지입니다.

세이프존 안에 있으면 벗어날 때까지 대기했다가 발동합니다.
