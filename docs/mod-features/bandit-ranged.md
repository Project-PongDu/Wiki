# 적대 NPC (원거리) — 50,000원

`bandit_ranged`

원거리 화기 위주로 무장한 NPC **2인** 클랜을 소환합니다. [근접형](bandit-melee.md)보다 인원은 적지만 저격(Sharpshooter) 전문성을 가진 개체가 포함되어 명중률과 사거리에서 이점을 갖습니다.

- 세이프존 안에 있으면 벗어날 때까지 대기했다가 소환됩니다.
- 스폰이 몰릴 경우 대기열에 쌓였다가 순서대로 처리됩니다.
- 후원자가 지정되어 있으면, 소환된 NPC 근처에 일정 시간 동안 "{후원자}의 자객" 형태의 표식이 표시됩니다.
