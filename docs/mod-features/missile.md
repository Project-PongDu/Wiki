# 미사일 폭격 — 150,000원

`missile`

지정 반경에 폭격을 가하는, 기본 티어 중 가장 파괴력 있는 기능입니다.

1. 발동 즉시 반경 40타일 내 플레이어들에게 경보음이 브로드캐스트됩니다.
2. 카운트다운 타이머가 화면에 표시됩니다.
3. 폭발 시:
     - 폭격 반경(`Donation_BombardRadius`, 5~60타일, 기본 55) 내의 좀비를 전부 처치합니다. B41 멀티플레이 특성상 좀비는 **접속 중인 클라이언트가 소유권을 가진 개체만** 정상적으로 죽일 수 있어, 각 클라이언트가 자신이 소유한 좀비만 처리합니다.
     - 폭발 시점에 반경 안에서 실외에 있던 플레이어는 피해를 입습니다 — 머리 화상, 신체 부위 찰과상·열상·깊은 자상이 무작위로 적용되고 넘어짐 처리됩니다.

!!! info "즉시 발동 / 반경은 강령술과 별개"

    세이프존 안에 있어도 대기 없이 바로 발동합니다. 폭격 반경(`Donation_BombardRadius`)과 [강령술](rise-up-dead-man.md) 반경(`Donation_RiseUpRadius`)은 이름이 비슷해 보이지만 서버 설정에서 완전히 독립된 변수입니다.
