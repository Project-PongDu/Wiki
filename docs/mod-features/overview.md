# 개요

t3chzzkDonation 모드는 퐁듀가 기록한 `rewards.txt`를 읽어서 후원 금액에 매핑된 featureId를 실행합니다. 이 섹션은 퐁듀 앱의 **기본 리워드 티어**에 배정된 11개 기능을 금액 순으로 설명합니다.

| --- | --- |
| 1,000원 | [버프 룰렛](buff-roulette.md) |
| 2,000원 | [디버프 룰렛](debuff-roulette.md) |

## 즉시 발동 vs 대기 발동

플레이어가 **세이프하우스로 등록된 구역**(및 주변 10타일) 안에 있으면, 대부분의 기능은 그 구역을 벗어날 때까지 5초 간격으로 대기했다가 발동합니다. 단, [백신](vaccine.md)과 [미사일 폭격](missile.md)은 세이프존 여부와 무관하게 즉시 발동합니다.

## 발동 순서

여러 후원이 동시에 들어와도 한 번에 하나씩 순서대로 처리됩니다. 대기 중인 이벤트가 있으면 큐에 쌓였다가 순서대로 실행됩니다.
