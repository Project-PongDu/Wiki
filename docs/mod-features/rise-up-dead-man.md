# 강령술 — 200,000원

`rise_up_dead_man`

플레이어 주변 반경 내의 좀비 시체를 모두 되살립니다. 기본 티어 중 가장 비싼 만큼 파급력도 큽니다.

- 발동 반경은 서버 설정(`Donation_RiseUpRadius`, 5~60타일, 기본 55)으로 조절되며, [미사일 폭격](missile.md)의 반경과는 완전히 별개 수치입니다.
- 시전 시 화면에 발동 범위가 시각적으로 표시됩니다.
- 되살아난 좀비가 원래 스프린터였다면 부활 후에도 스프린터 속도를 유지합니다.
- 되살아난 개체가 [특수좀비](mutant-spawn.md)였다면 종류·능력까지 그대로 복원되며, 서버가 재시작되어도 유지됩니다.

세이프존 안에 있으면 벗어날 때까지 대기했다가 발동합니다.
